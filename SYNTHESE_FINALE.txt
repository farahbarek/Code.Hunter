🔧 TECHNOLOGIES UTILISÉES
=========================

FRONTEND:
- HTML5 / CSS3 (Glassmorphisme + animations)
- JavaScript Vanilla (Fetch API)
- Google Fonts (Outfit, Space Mono, Syne)

BACKEND:
- Flask 1.x (Web framework)
- Flask-CORS (Cross-origin requests)
- joblib (Model serialization)
- pandas (Data manipulation)
- scikit-learn (Machine Learning)

ML MODEL:
- Random Forest Classifier
- 100 estimators
- max_depth = 15
- AUC-ROC ~ 0.82
- Accuracy ~ 75%
- 42 features (post one-hot encoding)

DATA:
- ~24,000 historical FBI homicide cases
- Features: Victim demographics, weapon type, location
- Target: Case resolved (binary classification)
